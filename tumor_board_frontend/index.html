Shared frontend invariants

Visible always:
  • RUO strip at root layout.
  • model_state chip on every claim-bearing block.
  • Non-dismissible Human Decision footer on every therapy view.
  • All data-ruo="true" attributes carried in DOM.
Never allowed:
  • Zero fake NCT numbers (NCT-EXAMPLE-*). All ClinicalTrials.gov references are real search links.
  • Zero placeholder receipts. Every p-value, SHA, and request_id comes from live transcripts.
  • Falsification never hidden behind "degraded" or "quarantine" tabs — always first-class row.
  • Manuscript-anchored blocks never carry proxy_* chips. Discovery-style ranking never claimed for audit sorts.

Component matrix at component_spec.json. The failing-claim strings enforced by quarantine_test_harness.json are the ones that must never render anywhere on the deliverable. Acceptance test output at acceptance_tests.json — see README.md for T01–T10 status.
